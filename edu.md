# Smart Rural Learning Hub

## Project Goal
Build an offline-first learning web application for rural students.

## Subjects
1. Math (Real PDFs and Videos)
2. English (Dummy PDFs and Videos)
3. Science (Dummy PDFs and Videos)

## Features Required

### 1. Login Page
- Name input
- Class selection (6thâ€“10th)

### 2. Dashboard
- Student Name
- Class
- Overall Progress
- Subject-wise Progress Bars
- Daily Streak
- Offline Indicator

### 3. Profile
- Name
- Class
- Progress
- Streak
- XP Points

### 4. Notes Section
- Subject-wise tabs
- Math has real PDFs
- English & Science have dummy PDFs
- Offline download support

### 5. Lecture Videos
- Math has real videos
- English & Science dummy videos

### 6. Roadmap
- Subject-wise chapter list
- Checkbox to mark completed
- Updates progress

### 7. AI Doubt Solver
- Predefined questions & answers
- Chatbot style UI

### 8. Quiz System
- Subject-wise quizzes
- After submission:
  - Show score
  - Show good topics
  - Show weak topics
  - Show motivational message

## Technical Requirements
- React + Vite
- Tailwind CSS
- Offline support using Service Worker
- Store progress using localStorage
- Colorful and attractive UI
